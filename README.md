# beTrader - Trading Journal PWA

A Progressive Web App (PWA) for tracking and analyzing your trading performance.

## Features

- Multi-account support with PIN protection
- Trade logging (Profit/Loss)
- Real-time statistics and analytics
- Trading history with PDF export
- Bilingual support (English/Indonesian)
- Offline-first functionality
- Swipe navigation

## Installation

### As PWA (Recommended)
1. Open the app URL in your browser
2. Click "Add to Home Screen" or install prompt
3. The app will work offline

### Local Development
1. Clone this repository
2. Open `index.html` in a browser
3. Or use a local server: `npx serve`
